Instalación del sistema operativo, hacer que parezca Abalar, configurar el WiFi e instalar Epoptes.

01
Instalar Abalinux 11
El flasheo de la imagen ISO debe hacerse desde un dispositivo con Windows.

Paso 1 — Descargar la imagen ISO

Descarga la imagen oficial de Abalinux desde el enlace:

Descargar Abalinux ISO

Paso 2 — Flashear el pendrive

Inserta un pendrive de mínimo 16 GB en el ordenador Windows y usa un programa como Balena Etcher para flashear la ISO en el pendrive.

Paso 3 — Arrancar desde el pendrive

Inserta el pendrive en el dispositivo donde quieres instalar Abalinux. Al encenderlo arrancará como Live USB directamente.

Paso 4 — Instalar

Una vez dentro del Live USB, ve a Aplicacións > Sistema > Instalador de Snapshot Live y sigue los pasos hasta que termine la instalación.

Cuando termine la instalación, quita el pendrive y reinicia. Abalinux quedará instalado en el ordenador.
02
Cambiar el fondo de pantalla al de Abalar

Pasos para reemplazar el fondo de pantalla por el fondo oficial de Abalar.

Paso 1 — Descargar el fondo

Descargar fondo de Abalar

Paso 2 — Renombrar el archivo

Renombra el archivo descargado de l0l612 a default (sin extensión).

Paso 3 — Reemplazar el fondo del sistema

Abre una terminal como administrador (su -) y ejecuta estos comandos:

bash — consola root
rm /usr/share/images/desktop-base/default
mv /home/usuario/Descargas/default /usr/share/images/desktop-base/
Después de ejecutar todos los comandos, el fondo se habrá cambiado.
03
Conectar al WiFi ABALAR — IES Valle Inclán

Pasos para conectarse a la red WiFi ABALAR del IES Valle Inclán mediante nmcli. Estos comandos eliminan la conexión guardada anterior (si existe) y crean una nueva con la contraseña correcta.

La contraseña es larga, cópiala exactamente tal como aparece. No la escribas a mano.

Contraseña de la red

contraseña WiFi
166f0b08dff0032f1224a6cd9b449d816fe295cf148ff6360327f1fa129c900c

Comandos para conectarse

Ejecuta estos comandos en orden, uno a uno:

bash — ejecutar en orden
nmcli connection delete "ABALAR"
nmcli connection add type wifi con-name "ABALAR" ssid "ABALAR"
nmcli connection modify "ABALAR" wifi-sec.key-mgmt wpa-psk
nmcli connection modify "ABALAR" wifi-sec.psk "166f0b08dff0032f1224a6cd9b449d816fe295cf148ff6360327f1fa129c900c"
nmcli connection up "ABALAR"
⚠️ El primer comando (connection delete) puede dar error si la red no existía antes — es normal, continúa con el siguiente.

Verificar la conexión

bash
nmcli connection show --active

01
Conseguir permisos de administrador

Este método accede al editor de arranque del sistema para obtener una consola de root sin contraseña. A partir de ahí se configura el usuario para tener acceso completo de administrador.

⚠️ Uso responsable. Este procedimiento otorga control total sobre el dispositivo.

Paso 1 — Entrar al editor de boot

  • En la pantalla de inicio (rEFInd), pulsa F2 para entrar a los ajustes.
  • Pulsa F2 de nuevo para abrir el editor de boot.
  • Usa la flecha abajo para ir al final de la línea.
  • Añade un espacio y escribe exactamente: init=/bin/bash
  • Pulsa Enter para arrancar.
Entrarás a una consola de root sin interfaz gráfica.

Paso 2 — Comandos en la consola de root

Ejecuta los siguientes comandos en orden.

bash — consola root (ejecutar en orden)
mount -o remount,rw /
chmod 4755 /bin/su
echo "usuario ALL=(ALL:ALL) NOPASSWD: ALL" > /etc/sudoers.d/usuario
echo "usuario ALL=(ALL:ALL) NOPASSWD: ALL" >> /etc/sudoers
usermod -s /bin/bash root
usermod -s /bin/bash usuario
passwd root
Cuando passwd root pida contraseña, escribe toor (dos veces).

Paso 3 — Apagar y reiniciar

Mantén el botón de apagado hasta que el equipo se apague. Al volver a encenderlo, abre una terminal y ejecuta:

bash
su -

Cuando pida contraseña escribe toor. Ya tendrás acceso a una consola de root con control total del dispositivo: puedes instalar, borrar y configurar cualquier cosa.

02
Desinstalar CTparental (control parental)

CTparental es el sistema de bloqueo de páginas web instalado en Abalinux. Desinstalarlo permite acceder a cualquier página web (Instagram, TikTok, etc.). Para hacerlo necesitas permisos de administrador primero (ver bloque anterior).

⚠️ Necesitas tener una consola de root abierta (su -) antes de ejecutar estos comandos.

Desactivar y desinstalar CTparental

Ejecuta estos comandos en orden desde la consola de administrador:

bash — consola root (ejecutar en orden)
sudo CTparental -off
sudo systemctl stop ctparental
sudo systemctl disable ctparental
sudo apt-get remove --purge ctparental
sudo apt-get autoremove
dpkg -l | grep -i parental
sudo apt-get remove --purge -y ctparental*
Después de ejecutar todos los comandos, el bloqueo de páginas habrá desaparecido.
